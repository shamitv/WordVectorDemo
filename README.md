# Word Vector Demo
Web application to demonstrate capabilities of Word Vectors. 

Hwoto : 

1. Clone the repo 
2. Run Springboot code in "WebApp
3. Modify "Config" class to point to a vector file (text/ csv)

Open http://localhost:8080/api/demo.html
